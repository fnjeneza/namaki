Namaki Client
=============

Getting started
---------------
Those packages are required to build namaki client

    * build-essential
    * cmake
    * ninja-build (optional)
    * zmq
    * protocol buffer

On ubuntu you can install them with:
    $ apt install build-essential cmake ninja-build libzmqpp-dev

For protobuf
